# Development Material

This directory contains curated examples, notebooks, and datasets used for:

* manual visual inspection
* stress testing
* edge-case validation
* development experiments

**This content is not part of the public API.**

Nothing in this directory is guaranteed to be stable, documented, or maintained 
for external use. Files may change, move, or be removed at any time without notice.

**Data sources**

1. Gavalda-Garcia, J. & Dixit, B. (**2024**)
Supplementary data frames, AlphaFold models, Normal Mode Analysis (NMA) Data, and NMA of Corresponding NMR Ensembles in the S2RCI, MD, and S2 Datasets for 
"*Gradations in protein dynamics captured by experimental NMR are not well represented by AlphaFold2 models and other computational metrics*".
DOI: [10.5281/zenodo.14192555](https://doi.org/10.5281/zenodo.14192555)
